Eigener Server

Ich habe meinen alten PC, welchen ich sowieso nicht mehr brauchte, zum Server umfunktioniert, um damit z.B. Game-Server oder Smart-Home Apps betreiben zu können. Da mir sehr am Herzen lag, dass ich auf den Server aus der nahen ferne zugreifen kann (mein momentaner PC ist ein Stockwerk über diesen) habe ich mich dafür entschieden casaOS auf diesen zu installieren.

Danach konnte ich den Server ganz einfach neben unserer Fritz!Box platzieren und die nötigen Apps installieren. Zu diesen zählt Crafty Controller. Crafty Controller ist die App mit welcher ich ganz einfach Minecraft Server für meine Freunde und mich aufsetzen kann.

Danach stellte sich noch die Frage wie man nun dem Server beitritt und da mir Port Forwarding etwas zu riskant war, habe ich playit.gg verwendet. Dies ist ein Service, welchen man als Plugin für den jeweiligen Minecraft Server installiert, auf welchen man dann einen einzigartigen Link bekommt, mit welchem man sich dann bei playit.gg einloggen kann. Dieser gilt quasi als Token, welcher diesen einen Server identifiziert, damit playit.gg genau weiß, mit welchem Server sie sich nun verbinden müssen. Und wenn alles geklappt hat, besteht nun ein Tunnel zwischen dem Server und playit.gg, sodass playit.gg quasi als VPN fungiert, welcher die Leute, mit der IP die auf dem Dashboard zugewiesen wurde, auf den Server weiterleitet.
